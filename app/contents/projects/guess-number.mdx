## Guess Number Game

The “Guess Number Game” is a simple yet engaging web-based game where players try to guess a randomly generated number within a specific range. The game provides instant feedback on whether the guess is too high or too low, helping players narrow down their choices until they find the correct number. With a clean and minimalistic design, this game is a great example of how basic web technologies like HTML, CSS, and JavaScript can be used to create interactive and fun applications.
