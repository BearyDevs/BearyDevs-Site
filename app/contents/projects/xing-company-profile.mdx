## Xing Smart Farm

Xing Smart Farm is a comprehensive company profile showcasing Xing’s advanced agricultural practices, focusing on cannabis farming. The platform highlights Xing’s innovative solutions, providing users with a detailed overview of their products and services, including indoor and outdoor farming technologies. Built with Next.js and styled with TailwindCSS, the site offers a seamless user experience, combining functionality with a modern design to reflect the cutting-edge nature of Xing’s farming solutions.
[Preview](https://xing-kingofcan-next-ts.vercel.app/)

### Stack

- [Next](https://nextjs.org/) - Next.js is a React framework for building full-stack web applications
- [TailwindCSS](https://tailwindcss.com/) - A utility-first CSS framework packed with classes that can be composed to build any design, directly in your markup.
