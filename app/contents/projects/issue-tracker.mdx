## Issue Tracker

The Issue Tracker is a free-time learning project designed to help developers efficiently manage and track issues within their applications. Built with Next.js, the project leverages a combination of modern technologies such as Prisma for database management, VercelDB for hosting the data, and NextAuth.js for handling authentication. The intuitive UI, styled with TailwindCSS, allows users to easily create, update, and assign issues, making it a great tool for developers and teams looking to streamline their project management processes.
[Preview](https://issue-tracker-toylab.vercel.app/)

### Stack

- [Next](https://nextjs.org/) - Next.js is a React framework for building full-stack web applications
- [VelcelDB](https://vercel.com/storage/postgres) - VercelDB is a fully managed, serverless SQL database optimized for use with modern web applications, providing scalable and performant data storage. Built on PostgreSQL, VercelDB offers automatic scaling, seamless deployment, and low-latency queries, making it a perfect choice for applications that require robust and efficient database management without the need for manual infrastructure handling.
- [TailwindCSS](https://tailwindcss.com/) - A utility-first CSS framework packed with classes that can be composed to build any design, directly in your markup.
- [PostgreSQL](https://hardhat.org) - Flexible, Extensible, Fast, Ethereum development environment for professionals
- [Prisma](https://www.prisma.io/) - ORM Tool to boost development speed with Prisma's powerful ORM and type safety features. Prisma offers real-time data syncing and seamless schema management.
- [Next-Auth](https://next-auth.js.org/) - NextAuth.js is an open source community project for implementing authentication and authorization in Next.js applications.
