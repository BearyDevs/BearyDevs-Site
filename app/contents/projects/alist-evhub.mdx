## Alist EVHUB

Car services mobile application, for users facilities like booking, test-drive, special offers, car specifications, and more in multibrands. Customers will fast to contract service centers and track services like repair, buy, financials, and more. Service centers have CRM for control and reports overall in the application

### Stack

- [Next](https://nextjs.org/) - Next.js is a React framework for building full-stack web applications
- [TailwindCSS](https://tailwindcss.com/) - A utility-first CSS framework packed with classes that can be composed to build any design, directly in your markup.
- [PostgreSQL](https://hardhat.org) - Flexible, Extensible, Fast, Ethereum development environment for professionals
- [Prisma](https://www.prisma.io/) - ORM Tool to boost development speed with Prisma's powerful ORM and type safety features. Prisma offers real-time data syncing and seamless schema management.
- [Next-Auth](https://next-auth.js.org/) - NextAuth.js is an open source community project for implementing authentication and authorization in Next.js applications.
